Donate Button¶

A quick and simple way to request donations on any page rendered inside ethers.io.

Any ethers.io enabled site can add a donate button, simply be including the JavaScript:

document.getElementById('your-donate-button').onclick = function() {
    ethers.send(yourAddress);
}
