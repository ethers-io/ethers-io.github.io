ethers.io Documentation
=======================

This documentation is generated using [Sphinx](http://www.sphinx-doc.org/) with the [Read The Docs](https://readthedocs.org) template.

Contributing
------------

To contribute, you must make updates to the documentation found at [ethers-io/docs](https://github.com/ethers-io/ethers-io/tree/master/docs).

```
> git clone git@github.com:ethers-io/ethers.io.git
> cd ethers.io/docs

... make changes ...

> make clean && make html
> open _build/html/index.html
```
